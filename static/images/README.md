# Carousel Images

## Where to Add Your Images

Place your carousel images in this folder: `static/images/`

## Image Requirements

### File Names:
- `carousel-1.jpg` (or `.png`) - For the first slide
- `carousel-2.jpg` (or `.png`) - For the second slide  
- `carousel-3.jpg` (or `.png`) - For the third slide

### Recommended Specifications:
- **Size**: 1920x1080px (Full HD) or larger
- **Aspect Ratio**: 16:9 (wide format)
- **Format**: JPG, PNG, or WebP
- **File Size**: Keep under 500KB per image for faster loading

### Alternative Names:
You can also use:
- `slide-1.jpg`, `slide-2.jpg`, `slide-3.jpg`
- `hero-1.jpg`, `hero-2.jpg`, `hero-3.jpg`
- Or any names you prefer (just update the template)

## How to Add Images

1. Copy your images from your device
2. Paste them into this folder: `static/images/`
3. Rename them to: `carousel-1.jpg`, `carousel-2.jpg`, `carousel-3.jpg`
4. Refresh your browser to see the changes

## Fallback

If images are not found, the carousel will automatically use gradient backgrounds as fallback.
